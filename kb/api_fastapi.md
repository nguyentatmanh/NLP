# FastAPI

## Vai trò
FastAPI là framework Python thường dùng để xây API hiệu năng tốt và dễ phát triển.

## Ứng dụng
Trong hệ QA, FastAPI có thể dùng để xây backend nhận câu hỏi, gọi retriever, reader và trả JSON cho giao diện web.

## Ưu điểm
Framework này hỗ trợ khai báo kiểu dữ liệu rõ ràng và tài liệu API tự động.
