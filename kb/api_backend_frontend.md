# API, Backend và Frontend

## Backend là gì
Backend là phần xử lý logic nghiệp vụ, dữ liệu, xác thực và giao tiếp với mô hình.

## Frontend là gì
Frontend là phần giao diện người dùng trực tiếp tương tác để nhập câu hỏi và xem câu trả lời.

## API là gì
API là giao diện kết nối giữa frontend và backend, cho phép truyền câu hỏi và nhận kết quả theo định dạng chuẩn.

## Vai trò của API trong hệ hỏi đáp
API giúp frontend gửi câu hỏi, backend gọi retriever và reader, rồi trả kết quả về giao diện.
