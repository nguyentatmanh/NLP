# Machine Learning

## Khái niệm
Machine Learning là phương pháp cho phép máy học từ dữ liệu để dự đoán hoặc phân loại.

## Dạng học
Các dạng học phổ biến gồm học có giám sát, không giám sát và học tăng cường.

## Ứng dụng
ML được dùng trong phân loại thư rác, dự báo, nhận dạng mẫu và hệ thống gợi ý.
