# Machine Learning và Deep Learning

## Machine Learning
Machine Learning là phương pháp cho phép máy tính học từ dữ liệu để đưa ra dự đoán hoặc quyết định mà không cần lập trình cứng toàn bộ quy tắc.

## Deep Learning
Deep Learning là một nhánh của Machine Learning, sử dụng mạng nơ-ron nhiều tầng để học biểu diễn dữ liệu phức tạp.

## So sánh
Machine Learning thường phù hợp khi dữ liệu có cấu trúc, số lượng vừa phải và cần mô hình dễ giải thích hơn. Deep Learning mạnh ở các bài toán dữ liệu lớn như ảnh, giọng nói và ngôn ngữ tự nhiên, nhưng thường cần tài nguyên tính toán cao hơn.
