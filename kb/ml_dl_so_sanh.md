# So sánh Machine Learning và Deep Learning

## Machine Learning là gì
Machine Learning là phương pháp cho phép máy tính học từ dữ liệu để đưa ra dự đoán hoặc quyết định mà không cần lập trình cứng toàn bộ quy tắc.

## Deep Learning là gì
Deep Learning là một nhánh của Machine Learning, sử dụng mạng nơ-ron nhiều tầng để học biểu diễn dữ liệu phức tạp.

## So sánh Machine Learning và Deep Learning
Machine Learning thường phù hợp với dữ liệu có cấu trúc và số lượng vừa phải, trong khi Deep Learning mạnh hơn ở các bài toán dữ liệu lớn như ảnh, âm thanh và ngôn ngữ tự nhiên. Deep Learning thường cần tài nguyên tính toán và dữ liệu lớn hơn.

## Ý nghĩa của việc so sánh
Việc so sánh hai hướng tiếp cận này giúp lựa chọn mô hình phù hợp với bài toán, dữ liệu và phần cứng hiện có.
