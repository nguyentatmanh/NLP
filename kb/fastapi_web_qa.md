# FastAPI trong hệ thống QA

## FastAPI là gì
FastAPI là framework Python dùng để xây dựng API nhanh, rõ ràng và hiệu quả.

## Vì sao chọn FastAPI
FastAPI dễ tích hợp với mô hình học máy, hỗ trợ kiểu dữ liệu rõ ràng và thuận lợi cho việc xây dựng hệ thống QA có giao diện web.

## Vai trò của FastAPI trong hệ thống
FastAPI đóng vai trò backend trung tâm, nhận câu hỏi, xử lý truy hồi và đọc hiểu, rồi trả phản hồi cho giao diện web.
